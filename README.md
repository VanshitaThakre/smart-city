# AI Autonomous Smart City Command Center

## Overview
AI Autonomous Smart City Command Center is a Streamlit-based dashboard that visualizes smart city data for different cities.

## Features
- Traffic Monitoring
- Water Management
- Safety Analysis
- Healthcare Dashboard
- Pollution Monitoring
- EV Charging Analysis

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
